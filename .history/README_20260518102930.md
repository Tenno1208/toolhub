## 🚀 Tech Stack & Fitur Unggulan

### ⚙️ Core Technologies
- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router & Turbopack)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Minimalist Dark Mode Aesthetics)
- **Component Icons:** [Lucide React](https://lucide.dev/)
- **AI Engine:** Google Gemini AI v1beta (Context-Aware AI Assistant)
- **Deployment:** [Vercel](https://vercel.com/) / [Netlify](https://netlify.com)

### 🧭 Eksplorasi 21 Fitur Utama (v2.0.0)
Aplikasi ini menyediakan antarmuka super responsif yang dibagi menjadi empat kategori utama:

1. **Developer & Security Tools:**
   - *JSON Formatter & Minifier* — Merapikan dan mengompresi struktur data objek.
   - *Base64 Converter* — Enkripsi dan dekripsi teks instan.
   - *SHA256 Hash Generator* — Pengacak keamanan berbasis Web Crypto API.
   - *Password Generator* — Generator sandi kuat dengan kustomisasi parameter variabel.
   - *SQL Query Formatter* — Merapikan indentasi syntax SQL otomatis (*New*).
   - *Markdown Live Editor* — Tulis markdown dengan split-screen preview (*New*).
   - *SVG to React JSX* — Transformasi tag SVG menjadi komponen fungsional Next.js (*New*).
   - *Git Command Generator* — Solusi baris perintah Git siap pakai (*New*).
   - *Regex Expression Tester* — Validasi kecocokan Regular Expression (*New*).

2. **Media & Utility:**
   - *QR Code Generator & Scanner* — Membuat dan memindai kode QR secara dua arah.
   - *Local Image Compressor* — Memperkecil ukuran file gambar 100% lokal.
   - *Color Picker* — Alat bantu ekstraksi kode warna HEX dan RGB untuk UI/UX.
   - *IP Info Finder* — Deteksi jaringan IP dan informasi provider secara aktual.
   - *TikTok Downloader* — Pengunduh video tanpa watermark dengan penamaan file unik.
   - *Instagram Downloader* — Engine pengunduh reels dan media beresolusi tinggi.

3. **Productivity & Essentials:**
   - *Portfolio Builder* — Generator web portofolio instan dengan live preview dan export HTML.
   - *To-Do List* — Manajemen tugas interaktif terintegrasi dengan `localStorage`.
   - *Text Case Converter & Word Counter* — Utilitas manipulasi dan penghitung instan teks.
   - *Stopwatch, Digital Calculator & Age Calculator* — Perhitungan presisi waktu, matematika, dan umur.

4. **Fun & Utilities (5 Tools Receh):**
   - *Teks Alay Generator* — Mengubah teks normal menjadi gaya warnet 2010.
   - *Keputusan Mutlak Picker* — Pengacak opsi keputusan interaktif berbasis delay acak.
   - *Emoji Mixer Studio* — Kombinator dua emoji berbeda menjadi satu representasi baru.
   - *Kalkulator Jodoh Gimmick* — Penghitung kecocokan asmara berdasarkan algoritma ASCII.
   - *Meme Soundboard* — Audio synthesizer lokal pemutar efek suara meme ikonik.

## 💻 Panduan Instalasi Lokal

```bash
# 1. Clone repositori ini
git clone [https://github.com/Tenno1208/toolhub.git](https://github.com/Tenno1208/toolhub.git)

# 2. Masuk ke direktori project
cd toolhub

# 3. Install semua dependensi package
npm install

# 4. Jalankan lokal development server
npm run dev
